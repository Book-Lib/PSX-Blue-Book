# Об этом руководстве

Это руководство является последней версией библиотеки для **Run-Time Library 4.6**. Целью данного руководства является предоставление обзорную информации о библиотеках **PlayStation®**. Соответствующие описания функций и структур **Run-Time Library PlayStation** можно найти в Справочнике по **Run-Time Library**.

## Изменения с момента последнего выпуска

Начиная с версии  **Run-Time Library 4.5**, были внесены следующие изменения:

**Глава 8: Основная графическая библиотека**

В раздел «Внимание по программированию» добавлен пункт «Сообщение о тайм-ауте графического процессора».

**Глава 11: CD / Streaming библиотека**

Внесены исправления в пункт «Пропущенные разделы» раздела «Примечания по использованию групп функций низкого уровня».

**Глава 19: PDA библиотека**

Добавлено примечание к элементу «Изображение значка выбора игры» таблицы 19-3 в разделе «Заголовок файла».
В пункт «Значки выбора игр» в разделе «Заголовок файла» добавлена дополнительная информация.
В раздел «Значки» добавлен пункт «Примечания к таблице ввода значков и расположению изображений значков».

# Сопутствующая документация

Это руководство следует читать вместе со Справочником по **Run-Time Library**, в котором определены все структуры и функции, доступные в **Run-Time Library**.

## Структура руководства

|Глава|Описание|
|-------|--------|
|Глава 1: Обзор ОС **PlayStation**|Краткое описание операционной системы **PlayStation** и библиотек времени выполнения.
|Глава 2: Kernel Library|Описывает библиотеку ядра (libapi), которая обеспечивает интерфейс между приложениями и ОС **PlayStation**.|
|Глава 3: Стандартная библиотека **C**|Описывает подмножество стандартной библиотеки **C** для **PlayStation (libc/libc2)**. Эта библиотека включает в себя символьные функции, функции операций с памятью, тесты классов символов, нелокальные переходы и служебные функции.|
|Глава 4: Математическая библиотека|Описывает математическую библиотеку (libmath), которая содержит математические функции, совместимые с ANSI/IEEE754, включая программный пакет вычислений с плавающей точкой.|
|Глава 5: Библиотека карт памяти|Описывает библиотеку карт памяти (libcard) для чтения/записи на карту памяти **PlayStation** и вызова службы **BIOS** карты памяти.|
|Глава 6: Расширенная библиотека карт памяти|Описывает расширенную библиотеку карт памяти (libmcard), которая обеспечивает интерфейс высокого уровня для доступа к карте памяти.|
|Глава 7: Библиотека сжатия данных|Описывает библиотеку сжатия данных (libpress) для сжатия (кодирования) и расширения (декодирования) изображений и звуковых данных.|
|Глава 8: Базовая графическая библиотека|Описывает базовую графическую библиотеку (libgpu) для рисования примитивов, таких как спрайты, многоугольники и линии.|
|Глава 9: Базовая библиотека геометрии|Описывает библиотеку базовой геометрии (libgte), которая использует сопроцессор PlayStation GTE для выполнения высокоскоростных геометрических операций.|
|Глава 10: Расширенная графическая библиотека|Описывает расширенную графическую библиотеку (libgs), которая использует libgpu и libgte для создания трехмерной графической системы. Она имеет дело с более абстрактными объектами, такими как объекты и фоновые поверхности.|
|Глава 11: Библиотека CD-ROM|Описывает библиотеку CD-ROM (libcd) для управления приводом **CD-ROM PlayStation**. Он также содержит функции для «потоковой передачи» данных в реальном времени, таких как фильмы, звуки или данные вершин, хранящиеся на носителях большой емкости.|
|Глава 12: Расширенная библиотека CD-ROM|Описывает расширенную библиотеку CD-ROM (libds), которая создает новый интерфейс для ядра libcd. Она имеет те же возможности, что и libcd, и уделяет особое внимание надежному управлению CD-ROM, например восстановлению после системных ошибок.|
|Глава 13: Библиотека контроллеров/периферийных устройств|Описывает библиотеку контроллера/периферийных устройств, состоящую из группы библиотек (libetc, libgun, libtap и libpad) для выполнения низкоуровневой обработки прерываний и функций, связанных с контроллером.|
|Глава 14: Библиотека Link кабель|Описывает библиотеку Link кабель (libcomb), которая предоставляет сервис по подключению устройств PlayStation через «линк-кабель».|
|Глава 15: Расширенная Звуковая библиотека|Описывает расширенную звуковую библиотеку (libsnd), предоставляющую функции для работы со звуковыми данными на PlayStation.|
|Глава 16: Основная звуковая библиотека|Описывает базовую звуковую библиотеку (libspu) для управления SPU (блоком обработки звука).|
|Глава 17: Библиотека последовательного ввода/вывода|Описывает библиотеку последовательного ввода-вывода (libsio) для поддержки связи между PlayStation и ПК.|
|Глава 18: HMD библиотека|Описывает библиотеку HMD (libhmd), которая предоставляет функции и определения для обработки формата HMD, который объединяет данные моделирования, анимации, текстуры и MIMe.|
|Глава 19: Библиотека PDA|Описывает библиотеку PDA (libmcx), используемую для обеспечения доступа к функциям PDA, когда он вставлен в слот карты памяти.|
|Глава 20: Модуль mcgui|Описывает модуль сохранения и загрузки данных в играх, а также поддержку пользовательского интерфейса.|

# Справочная серия для разработчиков

Это руководство является частью серии «Справочники для разработчиков» — серии технических справочных томов, охватывающих все аспекты разработки PlayStation. Полная серия представлена ниже:

|Руководство|Описание|
|-----------|--------|
|Оборудование PlayStation|Описывает аппаратную архитектуру PlayStation и делает обзор ее подсистем.|
|Операционная система PlayStation|	Описывает операционную систему PlayStation и связанные с ней основы программирования.|
|Run-Time Library|Описывает структуру и назначение Run-Time Library, предоставляемых для разработки программного обеспечения PlayStation.|
|Run-Time Library Ссылка|Определяет все доступные функции, макросы и структуры Run-Time Library PlayStation.|
|Справочник по встроенному программированию|Описывает встроенное программирование с использованием DMPSX, встроенного макроса GTE и информации о регистрах GTE.|
|Среда разработки SDevTC|Описывает среду разработки SDevTC (ранее «Psy-Q») для разработки программного обеспечения PlayStation.|
|Инструменты 3D-графики|Описывает, как использовать инструменты графики PlayStation 3D, включая редакторы анимации и материалов.|
|Редактор спрайтов|Описывает инструмент «Редактор спрайтов» для создания данных спрайтов и компонентов фонового изображения.|
Инструмент звукорежиссера|Содержит инструкции по установке и эксплуатации платы Sound Artist Board DTL-H800, а также объясняет, как использовать программное обеспечение Sound Artist Tool.|
|Форматы файлов|Описывает все собственные форматы данных PlayStation.|
|Утилиты преобразования данных|Описывает все доступные утилиты преобразования данных PlayStation, включая как автономные, так и подключаемые программы.|
|Эмулятор компакт-диска|Содержит инструкции по установке и эксплуатации подсистемы эмулятора компакт-дисков и соответствующего программного обеспечения.|
|Генератор компакт-дисков|Описывает, как использовать компакт-диск. Программа-генератор для записи дисков CD-R.|
|Технический анализатор производительности|Содержит общие инструкции по использованию программного обеспечения Performance Analyzer.|
|Технический анализатор производительности Ссылка|Описывает, как измерить программное обеспечение.производительность и интерпретировать результаты с помощью анализатора производительности.|
|Установка и эксплуатация DTL-H2000|Содержит инструкции по установке и эксплуатации системы разработки DTL-H2000.|
|DTL-H2500/2700 Установка и эксплуатация|Обеспечивает установку и эксплуатацию.инструкции для систем разработки DTL-H2500/H2700.|
